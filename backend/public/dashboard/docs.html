CloudPay Payment Gateway API

Create Payment

Endpoint:

POST /api/v1/payments

Headers:


x-api-key: test_key
x-api-secret: test_secret
idempotency-key: unique_key

Body:


{
 "amount": 100
}

Capture Payment


POST /api/v1/payments/{id}/capture

Create Refund


POST /api/v1/payments/{id}/refunds

Body:


{
 "amount": 50
}

Webhook Events

CloudPay sends webhooks when events occur.


payment.success
payment.failed
refund.processed
